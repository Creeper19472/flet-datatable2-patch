# flet-datatable2
DataTable2 control for Flet

## Installation

Add dependency to `pyproject.toml` of your Flet app:

```
dependencies = [
  "flet-datatable2 @ git+https://github.com/flet-dev/flet_datatable2.git",
  "flet>=0.27.4",
]
```

Build your app:
```
flet build macos -v
```
